# PlexAI - AI-Powered Customer Experience & Strategy Platform

**PlexAI** is a comprehensive AI-driven platform designed to optimize customer experience (CX), drive strategic decision-making, and maximize ROI through intelligent automation and ethical brand alignment.

## 🚀 Features

### 1. **Dashboard Overview**
- Real-time metrics on AI sprints, CX health, ethical alignment, and projected ROI
- Latest AI recommendations with automated response capabilities
- Active integrations status (Google Analytics, Salesforce, Shopify, etc.)

### 2. **CX Pulse**
- Real-time customer journey analytics with drop-off identification
- AI-powered personalization engine with dynamic A/B testing
- Track conversion funnel from homepage to checkout

### 3. **Strategy Sprints**
- Agile sprint management for marketing campaigns and customer retention initiatives
- AI-generated content variations and task tracking
- Real-time sprint progress monitoring

### 4. **Ethical Analyzer**
- Brand ethics scoring and alignment verification
- NLP-powered content scanning for authenticity and inclusivity
- Greenwashing detection and narrative adjustment suggestions

### 5. **ROI Command Center**
- Comprehensive ROI tracking and benchmarking
- KPI monitoring against industry goals
- Customer acquisition cost optimization metrics

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/H-E-Ingwee/plexai.git

# Navigate to project
cd plexai

# Install dependencies
npm install

# Start development server
npm run dev
```

## 📖 Component Structure

- `src/App.tsx`: Main application component with routing
- `DashboardView`: Executive summary dashboard
- `CXPulseView`: Customer experience analytics
- `StrategySprintsView`: Sprint management interface
- `EthicalAnalyzerView`: Brand ethics verification
- `ROICommandView`: ROI and KPI tracking
- Utility components: MetricCard, ProgressBar, TaskItem, ScanResult
